# CodeAlpha Calculator

A responsive calculator built using **HTML, CSS, and JavaScript** for the CodeAlpha Frontend Development internship.

## Features
- Basic arithmetic operations: `+`, `-`, `*`, `/`, `%`
- Clear (`AC`) and delete (`DEL`) buttons
- Keyboard support
- Responsive UI
- Light/Dark theme toggle

## Files
- `index.html`
- `style.css`
- `script.js`

## How to run
Open `index.html` in a browser.
